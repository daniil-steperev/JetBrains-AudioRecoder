# JetBrains-AudioRecoder

## Создайте Android-приложение на языке Kotlin, которое выполняет максимум требований из списка:

* позволяет записывать звук с микрофона;
* после окончания записи сохраняет полученный аудиофайл на Google Drive;
* каталог для сохранения указывается по ссылке в интерфейсе приложения.
